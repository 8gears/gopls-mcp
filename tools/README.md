# gopls-mcp

gopls-mcp is a Model Context Protocol (MCP) server that wraps the Go language server (`gopls`) to provide deep semantic context to AI agents. It enables fast, type-aware code navigation, discovery, and analysis across Go projects.

For detailed documentation, visit [gopls-mcp.org](https://gopls-mcp.org).
